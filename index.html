What I do

I started programming after an impulsive purchase of a starter kit for a microcontroller four years ago. The unexpected purchase made me see the incredible value and deep fascination for how thoughts could be structured into algorithms and code. It was fascinating for me to realize that different programming languages were better suited for different purposes, almost as if they provided users with different "powers". Emerging from the collective efforts of everyone investing their time on them.

Below is a quick overview of the tools I like to use: online resume and project portfolio.

Python

Awesome multitool I'm deeply passionate about. My first choice when solving a problem. From servers, databases, machine learning or just making plots, python accompanies me every step of the way.

Shell & Bash

I love bash and its ability to glue things together into a project for script making and job control.

Docker & Github

Creating version controlled repositores, developing docker images, and shipping them onto remote instances radically changed how I viewed things in the software world.

From navigating the world using Street View API from Google, setting up EC2 instances on AWS or applying AI solutions with Cognitive Services in Azure, I always rely on some form of cloud service for my projects.
