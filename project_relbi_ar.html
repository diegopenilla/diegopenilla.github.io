Relbi AR

Visualizing traffic data in mixed reality

Relbi AR

  • Industry: Transport / Tech
  • Team Size: 4
  • Website: www.devpost.com/jello
For the BVG Competition using the mixed reality toolkit on the Unity Engine 3D and a python based web server deployed on EC2. We created a mixed reality prototype app for the Hololens that displays information from BVG mobility partners after scanning station-specific QR codes.

What it does

  • A user wearing the Hololens scans a QR code (specific to a BVG station).
  • With the QR code, traffic data from different Jelbi Partners is processed by the web server.
  • The web server sends the data back to the Hololens for display

Challenges

The key problem that was solved was figuring out how to make requests using Unity, a cross-platform game engine. Solving this problem allowed us to bring data from the external world into our application. However, the data coming off from the traffic APIs had to be cleaned, understood and structured for a proper display in the Hololens. To solve this problem we created a simple python-based web server to act as mediator between the Hololens app and external traffic data.

By using hand gestures or scanning QR codes, requests get triggered inside the app, calling the web server and retrieving information specific to the gesture used or QR code scanned.

Tools and Services

Microsoft Hololens

An awesome piece of hardware with multiple sensors, cameras, advanced optics, and holographic processing technologies programmed using the mixed reality toolkit of the Unity Engine 3d.

Backend

Python web based server programmed using FastAPI, able to call some BVG mobility partners and Google Street View API to return text or images back to the hololens.

Deployment

The server was deployed as a Docker container on an AWS EC2 instance.

Traffic Data
The traffic data was provided by Jelbi, a BVG app that integrates into one application many mobility solutions in Berlin.
It would be interesting to try integrating additional AI services into the Hololens e.g to voice-command the application or recognize places automatically.
